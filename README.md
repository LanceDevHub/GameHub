# 🎮 Game Explorer

Eine einfache Webanwendung zum Durchsuchen und Filtern von Videospielen basierend auf der RAWG API.

---

## 🚀 Überblick

Der Game Explorer ermöglicht es, Spiele aus der Datenbank von RAWG.io zu durchsuchen und nach verschiedenen Kriterien zu filtern.
Die Anwendung dient als Frontend-Projekt zur Demonstration von API-Integration und UI-Design mit React.

---

## ✨ Funktionen

* Suche nach Spielen
* Filter nach verschiedenen Kriterien (z. B. Genre, Plattform, Bewertung)
* Anzeige von Spieldetails
* Dynamisches Laden von Daten über API

---

## 🧱 Tech Stack

* React
* JavaScript / TypeScript
* Vite (optional, je nach Setup)
* RAWG API

---

## 🌐 API

Die Daten stammen von:

👉 https://rawg.io/apidocs

Diese Anwendung nutzt die API ausschließlich als Datenquelle und ist stark an deren Plattform angelehnt.
